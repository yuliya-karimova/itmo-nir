## Детали изменения трафика BDUI

**Что добавилось в JSON ответ:**
- Описание структуры формы (title, description, submitText)
- Массив полей формы (fields) с их свойствами:
  - name, label, type, required для каждого поля
  - 5 предопределенных полей формы

**Преимущества BDUI:**
- Вся информация о форме приходит в одном запросе
- Не требуется пересборка фронтенда
- Форма может быть изменена через админ-панель без деплоя

**Недостатки BDUI:**
- Увеличение размера JSON ответа
- Больше данных для парсинга на клиенте
