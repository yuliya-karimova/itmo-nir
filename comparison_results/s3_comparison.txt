## Детали изменения трафика BDUI

**Что добавилось в JSON ответ:**
- Описание структуры формы (title, description, submitText)
- Массив полей формы (fields) с их свойствами:
  - name, label, type, required для каждого поля
  - 5 предопределенных полей формы

**Преимущества BDUI:**
- Вся информация о форме приходит в одном запросе
- Не требуется пересборка фронтенда
- Форма может быть изменена через админ-панель без деплоя

**Недостатки BDUI:**
- Увеличение размера JSON ответа (+15.3% при добавлении формы)
- Больше данных для парсинга на клиенте
- Небольшое увеличение использования памяти на клиенте (+0.3 MB)

**Преимущества Classic:**
- Не увеличивает размер API ответа при добавлении формы
- Лучшие показатели производительности (LCP, TTI, Speed Index)
- Меньше использование памяти на сервере
- Форма уже скомпилирована в бандл, не требует парсинга JSON

**Недостатки Classic:**
- Требуется пересборка фронтенда при изменении формы
- Форма не может быть изменена через админ-панель
- Небольшое увеличение времени Scripting (+6.0 мс)
- Увеличение времени Rendering и Painting из-за дополнительных DOM элементов
